# Text-to-Image Generator

This is a simple AI tool to generate images from text using the Stable Diffusion model.

## Requirements

- Python 3.8+
- PyTorch
- Transformers
- Diffusers

Install the required libraries using:

```bash
pip install torch torchvision transformers diffusers
